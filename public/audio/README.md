# Audio Files for Jazz Detective

Place your audio files in this folder. The game expects these tracks:

## Required Tracks

| Filename | Scene | Mood |
|----------|-------|------|
| `title-theme.mp3` | Title screen | Slow, moody jazz. Upright bass, brushed drums, muted trumpet. Sets the noir tone. |
| `tension.mp3` | Prologue | Suspenseful, minimal. Single piano notes, subtle strings. Building dread. |
| `club-ambient.mp3` | The Ember Room | Smoky club atmosphere. Background chatter, clinking glasses, distant jazz. |
| `dialogue-underscore.mp3` | Conversations | Subtle, low. Shouldn't distract from reading. Soft piano or bass. |
| `discovery.mp3` | Examining evidence | Slight tension, curiosity. Minor key, building interest. |
| `reflection.mp3` | Journal | Contemplative solo piano. Player is piecing things together. |

## Recommendations

- **Format:** MP3 or OGG (MP3 has best browser support)
- **Length:** 2-4 minutes, seamlessly loopable
- **Volume:** Normalize all tracks to similar levels
- **Style:** 1950s jazz or noir film score
